## ✨ 한 줄 소개
**1~2분짜리 레이스를 반복하며, 통계/기록을 쌓아 ‘내 선택’이 점점 똑똑해지는 경마 캐주얼 게임**.

---

## 🎮 게임 플레이
1. 4마리 말 중 한 마리를 고른다  
2. 배팅 코인을 정하고 확정한다  
3. 몬테카를로로 계산된 **승률/배당**을 확인한다  
4. 레이스를 관전한다  
5. 결과 정산 후 **리포트로 “왜 졌는지/이겼는지”** 확인한다  
6. 다시 도전하며 기록을 쌓는다  

---

## 🧩 핵심 특징
- **귀엽고 밝은 픽셀 아트 톤**
- 말 스탯(예: 속도/스태미나/운) 기반 레이스
- **몬테카를로 시뮬레이션 기반 승률/배당 표시**
- 결과를 납득하게 하는 **레이스 리포트(요약 + 이벤트 로그)**
- 1판 **1~2분** 내외, 하루 1시간 가볍게 반복 플레이에 최적

---

## 🕹️ 조작 (PC)
- `1~4` : 말 선택  
- `Enter` : 배팅 확정 / 레이스 시작  
- `R` : 결과 화면에서 다시 레이스  
- `Esc` : 뒤로 / 닫기  
- 마우스로도 말 카드/버튼 클릭 가능

---

## 💻 플랫폼
- **PC 우선**
- Android는 추후 업데이트 고려 (터치 친화 UI 방향 포함)

---

## 🛠️ 기술 스택
- Engine: **Godot 4.x**
- Save: 로컬 저장(`user://save.json` 등)

---

## 🧷 주의 사항 (중요)
- 이 게임은 **현금/실물화폐/환전 요소가 없습니다.**
- **가상 코인만 사용**하며, 순수 오락/통계 기반 선택 재미를 목표로 합니다.
